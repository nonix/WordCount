This is good old wc -w standard Unix utlity, just a bit faster than default.

Usage: wc <my_text_file>

Synopsis:
  The tool returns number of words separated by one or more: SP, TAB, NL
  
Environmet: 
  NTHREADS (optional) expects an integer as number of parallel threads. default is 16

To compile:
  make clean 
  make

Have fun!
